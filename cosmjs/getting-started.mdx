Get started with CosmJS in 60 seconds.

## 1. Install

```bash
npm install @cosmjs/stargate @cosmjs/proto-signing
```

## 2. Connect to a Chain

```typescript
import { StargateClient } from "@cosmjs/stargate";

const client = await StargateClient.connect("https://rpc.cosmos.network");

const chainId = await client.getChainId();
const height = await client.getHeight();
console.info(`Connected to ${chainId} at height ${height}`);
```

## 3. Query Data

```typescript
const balance = await client.getBalance("cosmos1...", "uatom");
console.info(`Balance: ${balance.amount}${balance.denom}`);
```

## 4. Create a Signer

```typescript
import { DirectSecp256k1HdWallet } from "@cosmjs/proto-signing";

const wallet = await DirectSecp256k1HdWallet.fromMnemonic("your mnemonic here ...", {
  prefix: "cosmos",
});
const [{ address }] = await wallet.getAccounts();
console.info("Address:", address);
```

## 5. Send a Transaction

```typescript
import { SigningStargateClient, GasPrice } from "@cosmjs/stargate";

const signingClient = await SigningStargateClient.connectWithSigner(
  "https://rpc.cosmos.network",
  wallet,
  { gasPrice: GasPrice.fromString("0.025uatom") },
);

const result = await signingClient.sendTokens(
  address,
  "cosmos1recipientaddress...",
  [{ denom: "uatom", amount: "1000000" }],
  "auto",
);
console.info("Tx hash:", result.transactionHash);
```

---

That's it. You've connected to a Cosmos chain, queried a balance, and sent
tokens — all in a few lines of TypeScript.
