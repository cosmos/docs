# Overview

CometBFT has some tools that are associated with it for:

- [Debugging](/cometbft/v0.38/docs/tools/debugging)
- [Benchmarking](#benchmarking)

## Benchmarking

- [https://github.com/informalsystems/tm-load-test](https://github.com/informalsystems/tm-load-test)

`tm-load-test` is a distributed load testing tool (and framework) for load
testing CometBFT networks.
