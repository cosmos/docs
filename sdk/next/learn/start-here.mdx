## Learn + Build (Recommended Path)

Get a solid mental model of the SDK and build your first chain.

1. Read the [intro pages](/sdk/next/learn/intro/overview) and [Cosmos Architecture](/sdk/next/learn/intro/sdk-app-architecture) for an overview of how a Cosmos chain is structured
2. Read the [Concepts](/sdk/next/learn/concepts/accounts) section (Fundamentals, Modules, SDK Internals)
3. Follow the [Build a Chain Tutorial](/sdk/next/tutorials/example/00-overview)

This is the most complete path for developers new to the Cosmos SDK.

## Get running fast

Spin up a local chain in minutes.

1. [Prerequisites](/sdk/next/tutorials/example/01-prerequisites)
2. [Chain Quickstart](/sdk/next/tutorials/example/02-quickstart)

From there, continue the tutorial to learn more about building modules:

- [Build a Module from Scratch](/sdk/next/tutorials/example/03-build-a-module): write your first custom module with messages, queries, and state
- [Full Counter Module Walkthrough](/sdk/next/tutorials/example/04-counter-walkthrough): add advanced features like fees, events, and module accounts
- [Run, Test, and Configure](/sdk/next/tutorials/example/05-run-and-test): test your chain and configure it for different environments
