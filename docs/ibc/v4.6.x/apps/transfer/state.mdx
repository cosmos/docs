The IBC transfer application module keeps state of the port to which the module is binded and the denomination trace information as outlined in [ADR 001](/docs/ibc/v4.6.x/architecture/adr-001-coin-source-tracing).

* `Port`: `0x01 -> ProtocolBuffer(string)`
* `DenomTrace`: `0x02 | []bytes(traceHash) -> ProtocolBuffer(DenomTrace)`
