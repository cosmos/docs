## Pre-Genesis Setup

<Warning>
  These parameters must be set **before** running `evmd init`. They cannot be
  changed after genesis without a hard fork.
</Warning>

- [ ] [Fork repository and rename from `evmd`](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#forking-and-initial-setup)
- [ ] [Set Bech32 address prefix](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#bech32-address-prefix) in `config/config.go`
- [ ] [Set BIP44 coin type](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#hd-wallet-configuration) in `crypto/hd/hdpath.go`
- [ ] [Choose token decimal precision](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#token-economics-configuration)
  - [ ] **If 18 decimals:** Uses standard bank module (default)
  - [ ] **If 6 or other decimals:** [Add precisebank module](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#adding-precisebank-module-for-non-18-decimal-chains) ([PreciseBank Module Docs](/evm/next/documentation/cosmos-sdk/modules/precisebank))



## Genesis Configuration

Run `evmd init <moniker> --chain-id <chain-id>` then edit `~/.evmd/config/genesis.json`:

### Chain Identity

- [ ] [Cosmos Chain ID](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#chain-ids) (string, e.g., `yourchain-1`)
- [ ] [EVM Chain ID](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#chain-ids) (number, register at [chainlist.org](https://chainlist.org))
- [ ] [Genesis time](https://github.com/cosmos/evm/blob/main/local_node.sh#L210) for coordinated launch

### Token Configuration

<Tip>
  **Working Example:** See [local_node.sh line 241](https://github.com/cosmos/evm/blob/296b5ea0227dfdc602488f97267567c4bf906ca1/local_node.sh#L241) for the exact bank metadata configuration used in the default evmd setup. This line shows the complete 18-decimal token configuration that works out-of-the-box.

  For the full genesis setup context, see [lines 234-248](https://github.com/cosmos/evm/blob/main/local_node.sh#L234-L248).

  For ERC20 module configuration and Single Token Representation, see the [ERC20 Module Documentation](/evm/next/documentation/cosmos-sdk/modules/erc20).
</Tip>

- [ ] [Bank metadata configuration](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#token-economics-configuration)
  - [ ] Base denom (`atest` for 18 decimal, `utest` for 6 decimal)
  - [ ] Display denom and exponent (18 or 6)
  - [ ] Token name, symbol, description
- [ ] [VM params: `evm_denom`](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#token-economics-configuration) (must match bank metadata base)
- [ ] **If non-18 decimals:** [Add `extended_denom_options`](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#token-economics-configuration)

## Runtime Configuration

<Warning id="validator-security-warning">
**Validator Security:** Validator nodes should NOT expose RPC/API ports publicly. Only RPC nodes (non-validators) should have these ports open.

See the [Validator Nodes Guide](/docs/evm/next/documentation/getting-started/node-configuration#validator-nodes) for full requirements.

**Exposing validator RPC ports can lead to:**
• DDoS attacks affecting block production
• Resource exhaustion and downtime with potential slashing
• Security vulnerabilities
</Warning>

### For RPC Nodes Only

Edit `~/.evmd/config/app.toml` on each RPC node:

- [ ] [Configure JSON-RPC](/docs/evm/next/documentation/getting-started/node-configuration#json-rpc) (enable, addresses, API namespaces)
- [ ] [Configure EVM settings](/docs/evm/next/documentation/getting-started/node-configuration#evm) (tracer, gas limits)
- [ ] [Configure Mempool settings](/evm/next/documentation/cosmos-sdk/modules/mempool-integration) (min-tip, block gas limit)
- [ ] [Configure TLS](/docs/evm/next/documentation/getting-started/node-configuration#application-config) (optional, see TLS section in app.toml for HTTPS)

### For All Nodes

- [ ] [Configure minimum gas prices](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#coordinated-network-launch)
- [ ] [Set persistent peers](/docs/evm/next/documentation/getting-started/build-a-chain/building-your-chain-guide#configure-persistent-peers)
