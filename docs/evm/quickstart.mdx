# Quick Start Guide

Get up and running with Cosmos EVM in just a few minutes. This guide will walk you through deploying your first smart contract.

## Prerequisites

Before we begin, ensure you have:

- **Node.js** (v18 or later)
- **Git** for version control
- **A code editor** (VS Code recommended)
- **MetaMask** or another Ethereum wallet

## Step 3: Write Your Smart Contract

Create a simple ERC20 token contract:

```solidity contracts/MyToken.sol
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;

import "@openzeppelin/contracts/token/ERC20/ERC20.sol";
import "@openzeppelin/contracts/access/Ownable.sol";

contract MyToken is ERC20, Ownable {
    constructor(address initialOwner)
        ERC20("My Token", "MTK")
        Ownable(initialOwner)
    {
        _mint(msg.sender, 1000000 * 10 ** decimals());
    }

    function mint(address to, uint256 amount) public onlyOwner {
        _mint(to, amount);
    }
}
```

## Step 5: Deploy Your Contract

### Compile Your Contract

```bash
npx hardhat compile
```

### Create a Deployment Script

```javascript scripts/deploy.js
const { ethers } = require("hardhat");

async function main() {
  const [deployer] = await ethers.getSigners();
  
  console.log("Deploying contracts with the account:", deployer.address);
  console.log("Account balance:", (await deployer.provider.getBalance(deployer.address)).toString());

  const MyToken = await ethers.getContractFactory("MyToken");
  const myToken = await MyToken.deploy(deployer.address);

  await myToken.waitForDeployment();

  console.log("MyToken deployed to:", await myToken.getAddress());
}

main()
  .then(() => process.exit(0))
  .catch((error) => {
    console.error(error);
    process.exit(1);
  });
```

### Deploy to Testnet

```bash
npx hardhat run scripts/deploy.js --network cosmosTestnet
```

You should see output similar to:


## Step 6: Interact with Your Contract

### Using Hardhat Console

```bash
npx hardhat console --network cosmosTestnet
```

```javascript
// In the console
const MyToken = await ethers.getContractFactory("MyToken");
const myToken = MyToken.attach("YOUR_CONTRACT_ADDRESS");

// Check token details
await myToken.name();        // "My Token"
await myToken.symbol();      // "MTK"
await myToken.totalSupply(); // 1000000000000000000000000

// Check your balance
const [owner] = await ethers.getSigners();
await myToken.balanceOf(owner.address);
```

### Using a Frontend

You can also interact with your contract using a simple HTML page:

```html index.html
<!DOCTYPE html>
<html>
<head>
    <title>My Token DApp</title>
    <script src="https://cdn.ethers.io/lib/ethers-5.7.2.umd.min.js"></script>
</head>
<body>
    <h1>My Token DApp</h1>
    <button onclick="connectWallet()">Connect Wallet</button>
    <button onclick="getBalance()">Get Balance</button>
    <div id="balance"></div>

    <script>
        const contractAddress = "YOUR_CONTRACT_ADDRESS";
        const contractABI = [
            "function name() view returns (string)",
            "function symbol() view returns (string)",
            "function balanceOf(address) view returns (uint256)",
            "function transfer(address to, uint256 amount)"
        ];

        let provider, signer, contract;

        async function connectWallet() {
            if (window.ethereum) {
                provider = new ethers.providers.Web3Provider(window.ethereum);
                await provider.send("eth_requestAccounts", []);
                signer = provider.getSigner();
                contract = new ethers.Contract(contractAddress, contractABI, signer);
                console.log("Connected!");
            }
        }

        async function getBalance() {
            if (contract && signer) {
                const address = await signer.getAddress();
                const balance = await contract.balanceOf(address);
                document.getElementById("balance").innerText = 
                    `Balance: ${ethers.utils.formatEther(balance)} MTK`;
            }
        }
    </script>
</body>
</html>
```

## Troubleshooting

### Common Issues

**Network Connection Errors**
- Verify your RPC URL is correct
- Check if you have sufficient testnet tokens
- Ensure your wallet is connected to the right network

**Gas Estimation Failures**
- Try increasing the gas limit in your transaction
- Check if your contract has any require statements that might fail

**Compilation Errors**
- Ensure you're using compatible Solidity versions
- Check for missing imports or dependencies
